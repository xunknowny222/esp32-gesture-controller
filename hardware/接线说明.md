## 📋 概述

ESP32-S3-EYE 是一款集成度很高的开发板，大部分外设已经集成在板子上，**无需额外接线**即可使用。

## 🔧 外部接线（可选）

### 如果需要连接外部设备：

### 1. 外部 OLED 显示屏（可选）

如果你不使用板载 LCD，可以连接外部 OLED：

| OLED 引脚 | ESP32-S3-EYE | 说明 |
|----------|-------------|------|
| VCC | 3.3V | 电源正极 |
| GND | GND | 电源负极 |
| SCL | GPIO 15 | I2C 时钟 |
| SDA | GPIO 4 | I2C 数据 |

**接线图**：
```
OLED 模块          ESP32-S3-EYE
┌─────────┐       ┌─────────────┐
│ VCC     │──────▶│ 3.3V        │
│ GND     │──────▶│ GND         │
│ SCL     │──────▶│ GPIO 15     │
│ SDA     │──────▶│ GPIO 4      │
└─────────┘       └─────────────┘
```

**代码示例**：
```cpp
#include <Wire.h>
#include <Adafruit_SSD1306.h>

#define SCREEN_WIDTH 128
#define SCREEN_HEIGHT 64
#define OLED_RESET    -1
#define SDA_PIN 4
#define SCL_PIN 15

Adafruit_SSD1306 display(SCREEN_WIDTH, SCREEN_HEIGHT, &Wire, OLED_RESET);

void setup() {
  Wire.begin(SDA_PIN, SCL_PIN);
  display.begin(SSD1306_SWITCHCAPVCC, 0x3C);
}
```

### 2. 外部传感器（可选）

#### I2C 传感器（如温湿度、加速度计）

| 传感器引脚 | ESP32-S3-EYE | 说明 |
|----------|-------------|------|
| VCC | 3.3V | 电源 |
| GND | GND | 地 |
| SCL | GPIO 15 | I2C 时钟（共享）|
| SDA | GPIO 4 | I2C 数据（共享）|

**注意**：I2C 总线可以并联多个设备，每个设备需要不同的地址。

---

#### SPI 传感器（如 RFID、SD 卡）

| 传感器引脚 | ESP32-S3-EYE | 说明 |
|----------|-------------|------|
| VCC | 3.3V | 电源 |
| GND | GND | 地 |
| SCK | GPIO 12 | SPI 时钟 |
| MISO | GPIO 13 | SPI 输入 |
| MOSI | GPIO 11 | SPI 输出 |
| CS | GPIO 10 | 片选（可自定义）|

---

### 3. 锂电池（可选）

ESP32-S3-EYE 支持外接锂电池：

**电池规格**：
- 电压：3.7V
- 容量：>1000mAh
- 类型：锂聚合物电池（带保护板）

**接线方式**：
- 焊接到板子背面的电池焊点
- 正极（+）：B+
- 负极（-）：B-

⚠️ **警告**：
- 焊接时注意正负极不要接反
- 使用带保护板的电池
- 充电时不要使用电池

## ⚠️ 注意事项

### 1. 电压警告

- **不要使用 5V 直接连接 GPIO** - 会烧坏芯片
- GPIO 工作电压：**3.3V**
- 最大耐受电压：**3.6V**

### 2. 电流限制

- 每个 GPIO 最大电流：**12mA**
- 所有 GPIO 总电流：**<200mA**
- 驱动大电流设备请使用三极管或 MOSFET

### 3. I2C 总线

- SDA 和 SCL 需要**上拉电阻**（板载已有）
- 总线长度：**<30cm**
- 设备数量：**<10 个**

### 4. 摄像头排线

- **不要频繁插拔** - 容易损坏
- 如果必须插拔，先断电
- 插紧后轻轻拉一下确认
