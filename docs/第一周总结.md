## ⚠️ 遇到的问题与解决

### 问题 1: ESP-IDF 组件下载失败

**现象**：
```
ERROR: Error during request URL: 403 Forbidden
```

**原因**：
- 网络连接问题
- ESP-IDF 组件服务器访问受限

**解决方案**：
- 改用 Arduino IDE 进行开发
- ESP-IDF 作为备选方案

**状态**: ✅ 已解决

**经验**：国内网络环境下，Arduino IDE 更稳定

---

### 问题 2: PSRAM 配置错误

**现象**：
```
frame buffer malloc failed
```

**原因**：
- 未选择正确的 PSRAM 类型

**解决方案**：
- Arduino IDE 中选择 `OPI PSRAM`
- 确认 PSRAM 容量为 8MB

**状态**: ✅ 已解决

**经验**：ESP32-S3-EYE 必须使用 OPI PSRAM

---

### 问题 3: 摄像头 FB-OVF 错误

**现象**：
```
cam_hal: FB-OVF
```

**原因**：
- 分辨率过高，内存不足
- 帧缓冲区溢出

**解决方案**：
- 降低分辨率到 QQVGA (160×120)
- 使用灰度图格式（PIXFORMAT_GRAYSCALE）
- 减少帧缓冲区数量为 1

**状态**: ✅ 已解决

**经验**：先保证功能可用，再优化性能

---

### 问题 4: 串口无输出

**现象**：上传成功后串口监视器无显示

**原因**：
- 波特率不匹配
- USB 模式配置问题

**解决方案**：
- 确认波特率为 115200
- 启用 `USB CDC On Boot`
- 上传后按 RESET 键

**状态**: ✅ 已解决

**经验**：串口调试需要正确配置 USB 模式
